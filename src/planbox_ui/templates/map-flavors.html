Bike Parking Map

Demo →

Make it easy and social for people to request bike parking. Collect suggestions, including photos of the sidewalk to help evaluate the locations. Use the admin interface to review and respond to requests, and to see stats—like which locations are popular.

  • Easily collect & manage requests for bike parking.
  • Engage people with address lookup for easy location finding on desktop or mobile.
  • Follow up with confirmation emails to participants.
  • Track activity with daily email reports.
  • Get the complete picture with the OpenPlans Dashboard.
  • Location of Request via Map Pin (data saved: latitude/longitude, approximate address via geolocation)
  • Location Name
  • Location Description
  • Submitter's Name (optional)
    or Name & Avatar (if signed in via Twitter/Facebook)
  • Email Address (private & optional)
  • ZIP Code (private & optional)
  • Age (private & optional)
  • Race/Ethnicity (private & optional)
